
# sleep-time

sleep-time to sleep the specified milliseconds.

# Usage

Install the library with `npm install sleep-time`

```javascript
var sleepTime = requre('sleep-time');

# second sleep
sleepTime(1000);
```


**ES6**

```javascript
let sleepTime from 'sleep-time';
```
